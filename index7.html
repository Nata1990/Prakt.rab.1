Задача 1: Решите квадратное уравнение:


3x2-21x+36=0


Решение:

Квадратное выражение имеет вид:
ax2+ bx+c=0
Дискриминант D равен:
D = b2 - 4ac = (-21)2 - 4 * 3 * 36 = 9;
x1 =(-b + √ D) / 2a = (-(-21) + √ 9) / 2 * 3 = 4;
x2 =(-b - √ D) / 2a = (-(-21) - √ 9) / 2 * 3 = 3;

Ответ: x1 = 4; x2 = 3.


Задача 2: Сколько молекул атомов каждого химического элемента содержится в веществе?


С2O2(OH)

Ответ:

В веществе C2O2(OH)содержится:
- 2 атома углерода (C);
- 3 атома кислорода (O);
- 1 атом водорода (H).
